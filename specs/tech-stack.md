# Tech Stack

- **Backend Framework:** Node.js with **TypeScript** and **Express.js** for building RESTful APIs.
- **Frontend Framework:** React supported by **Next.js** (using TypeScript) for building the user interfaces.
- **Database:** **SQLite** for a lightweight, file-based relational database.
- **Testing/Validation:** **Vitest** for fast and reliable unit/integration testing.
- **UI/UX:** **Responsive Design** approach from the ground up for mobile and desktop support.
- **Goal:** Provide a reliable, popular full-TypeScript stack with a seamless developer experience.
